TO create Servlet

create a "Java Package" in sourcePackage folder ie. "com.servlets"

Implement all the method of interface Servlets after importing "import javax.servlet.*;"

Creating web.xml file by right-clicking on the folder 'WebPages/WEB-INF/' and select Standard Deployment Descriptor or you can select other and type web.xml

Add the tags - commented in the index.html file.
